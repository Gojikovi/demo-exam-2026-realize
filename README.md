# 🧪 Демонстрационный экзамен 2026

## Сетевое и системное администрирование (09.02.06)

---

## 📌 О проекте

Данный репозиторий посвящён созданию домашнего стенда и полной реализации демонстрационного экзамена 2026 года. 

Если не грядут большие изменения в заданиях - подойдёт для последующих годов (2027, 2028 и т.д.).

Проект включает:

* разбор заданий
* пошаговую реализацию модулей
* конфигурации
* практические примеры

📎 Репозиторий создан обычным студентом колледжа, заканчивающим обучение по специальности **сетевого и системного администратора**, с целью упростить подготовку к экзамену как для себя, так и для других.

---

## 🎯 Цель

Помочь студентам подготовиться к демонстрационному экзамену через практику и реальную настройку инфраструктуры.

---

## 🧰 Что будет в репозитории

* 🔧 Развёртывание лабораторного стенда
* 🌐 Настройка сетевой инфраструктуры (Модуль 1)
* 🖥 Организация сетевого администрирования (Модуль 2)
* 📦 Готовые конфигурации
* 📸 Скриншоты выполнения
* 📄 Примеры заполнения отчётов

---

## 🛠 Используемые инструменты и ПО

* VMware Workstation
* Proxmox VE
* EcoRouterOS (в качестве ОС 2-ух маршрутизаторов)


📌 В рамках проекта 2 маршрутизатора будут реализованы на базе EcoRouterOS, вместо альтернативных решений (например, Alt JeOS), указанных в официальной документации.

---

🟢 Завершён (Возможны доработки M1/M2-speed-no-extra.md)

## 📂 Структура проекта

```
realization   - развёртывание стенда и способы реализации
images        - ссылки на скачивание образов операционных систем и Additional.iso
pictures-r    - скриншоты процесса развёртывания  
pictures-m1   - скриншоты выполнения модуля 1  
pictures-m2   - скриншоты выполнения модуля 2  
docs          - официальная документация экзамена  
configs       - конфигурации с пояснениями, комменариями и speed-no-extra конфиги
mydocs        - личные заметки, вспомогательные файлы, таблицы IP и доп. материалы  
```
